# Task 4: Advanced Encryption Tool

## Description
This task involves developing a secure encryption and decryption tool using
AES-256 encryption. The tool protects files using a password-based key.

## Algorithm Used
- AES-256 (Advanced Encryption Standard)

## Features
- Encrypt files securely
- Decrypt encrypted files
- Password-based protection

## Concepts Used
- Cryptography
- File handling
- Secure key generation
